# Debug Screenshots

This folder stores browser screenshots captured by agents during Playwright MCP tasks.

## Convention

- Create one subfolder per task using a short kebab-case description.
- Keep screenshots in chronological order with numbered filenames.
- Example:

```text
debug-screenshots/
  add-item-to-cart/
    01-homepage.png
    02-product-open.png
    03-cart-updated.png
```

These screenshots are intended for workshop demos so the browser actions are easy to show and review.
